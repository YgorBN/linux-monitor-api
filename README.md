# Linux Monitor API

API de monitoramento de sistema desenvolvida com Python e FastAPI.

## Funcionalidades

- Monitoramento de uso da CPU
- Monitoramento de memória RAM
- Monitoramento de disco
- Monitoramento de rede
- Exibição do horário em que o sistema foi iniciado

## Tecnologias utilizadas

- Python
- FastAPI
- Uvicorn
- psutil
- Git
- GitHub
